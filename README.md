# Student Management System

A full-stack student management system built with PHP and SQLite.  
This project demonstrates admin authentication, database handling, and CRUD operations.

## Features
- Admin login with session-based authentication
- Add new students
- View student list
- Delete students
- Secure password hashing
- Persistent SQLite database
- Sample student records for demonstration

## Technologies Used
- PHP
- SQLite
- HTML
- CSS
- PDO (PHP Data Objects)

## System Flow
1. Admin logs in
2. Admin can add students
3. Students are displayed on the dashboard
4. Admin can delete students
5. Logout ends the session
